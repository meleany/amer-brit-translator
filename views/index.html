American-British English Translator

FreeCodeCamp Quality Assurance Project 05

User Stories

  1. I can enter a simple sentence into the text area and select to translate to British or American English from the dropdown menu.
  2. When the "Translate" button is pressed, the translated sentence is appended to the div translated-sentence. The JavaScript files in /public contain the different spelling and terms the application should translate.
  3. The application should handle the way time is written in American and British English. For example, ten thirty is written as "10.30" in British English and "10:30" in American English.
  4. The application should handle the way titles/honorifics are abbreviated in American and British English. For example, Doctor Wright is abbreviated as "Dr Wright" in British English and "Dr. Wright" in American English. The file /public/american-to-british-titles.js contains the different titles the application should handle.
  5. Wrap any translated spelling or terms with <span class="highlight">...</span> tags so they appear in green.
  6. If the sentence in the text area has no spelling or terms that should be translated, append the message "Everything looks good to me!" to the div translated-sentence.
  7. If the text area is empty, append the message "Error: No text to translate." to the div error-msg so the text appears in red.
  8. I can press the "Clear Input" button to remove all text from the text area and the translated-sentence div.
  9. All 20 unit tests are complete and passing. The file /tests/1_unit-tests.js contains the sentences that should be tested.
  10. All 4 functional tests are complete and passing. Details for the functionality to be tested are given in /tests/2_functional-tests.js.
